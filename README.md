# Calculator-App-cpp
A simple and interactive console-based calculator application built in C++ supporting addition, subtraction, multiplication, and division.

A simple **console-based calculator application** built using **C++**.  
This program allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division.

---

##  Features
- Addition (+)  
- Subtraction (-)  
- Multiplication (*)  
- Division (/) with zero-division check  
- Simple user-friendly console interface
